## 项目结构

my_blog/services/my-blog/

├── src/
│   ├── controllers/     # 控制器 - 处理请求和响应
│   ├── services/        # 服务层 - 业务逻辑处理
│   ├── models/          # 数据模型 - 数据库交互
│   ├── routes/          # 路由定义
│   ├── middleware/      # 自定义中间件
│   ├── utils/           # 工具函数
│   ├── config/          # 配置文件
│   └── types/           # TypeScript 类型定义
├── tests/               # 测试文件
├── public/              # 静态文件
├── sql/                 # sql 脚本
└── docs/                # 项目文档
